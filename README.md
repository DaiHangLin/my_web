npm run build
-

+ 用 CLI 这种方式来运行本地的 webpack 不是特别方便，在 package.json 添加一个 npm 脚本(npm script)

        {
          ...
          "scripts": {
            "build": "webpack"
          },
          ...
        }

+ 可以使用 npm run build 命令，来替代我们之前用到的较长命令

        npm run build

+ [npm run build README.md](https://github.com/DaiHangLin/my_web/blob/npm/README.md)
